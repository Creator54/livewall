# Livewall
A simple live wallpaper setter for linux written in bash 

#### Dependencies :
* ffmpeg
* feh

#### Usage :
```
wget https://raw.githubusercontent.com/Creator54/livewall/main/livewall
./livewall use vid.mp4 
./livewall use vid.mp4 png 20
```
